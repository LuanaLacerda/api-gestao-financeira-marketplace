# 📊 API de Gestão de Marketplace e Fluxo Financeiro

Este projeto é uma solução de Back-end completa para o gerenciamento de um marketplace, permitindo o controle de usuários, produtos e transações financeiras. Desenvolvido originalmente durante minha jornada na **Cubos Academy**, ele hoje serve como base para estudos de **Modelagem de Dados** e **Insights de Negócio**.

### 💼 Contexto de Negócio (Foco em Resultados)
Como **Representante de Negócios na Ambev**, entendo que a integridade dos dados na ponta da venda é o que permite uma análise de mercado precisa. Este sistema simula a estrutura real de um PDV (Ponto de Venda) e Marketplace, lidando com:
*   **Gestão de Estoque/Produtos:** Organização por categorias.
*   **Transações Financeiras:** Registro de fluxos de entrada e saída.
*   **Segurança de Dados:** Autenticação e integridade das informações.

### 🛠️ Tecnologias e Dados
*   **Node.js & Express:** Lógica de negócio e roteamento.
*   **PostgreSQL:** Banco de dados relacional robusto para armazenamento de transações.
*   **SQL Estruturado:** Consultas para extração de dados e validação de saldos.

### 🔍 Visão de Analista de Dados
Diferente de uma visão puramente de desenvolvimento, a análise deste projeto permite responder KPIs como:
1. **Ticket Médio por Produto:** Qual a média de valor das transações realizadas?
2. **Saúde Financeira:** Balanceamento entre entradas e saídas por período.
3. **Curva de Produtos:** Quais categorias possuem maior movimentação no banco de dados?
